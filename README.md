# GAB Code Syntax Highlighter Extension 
Extension made to support syntax highlighting for GAB language to make reading code within Cursor more readable. 

# Install Instructions 
Currently, the best way to install this extension is by downloading the .vsix file from this repo and then installing it into either cursor or vscode from the terminal. 
1. Download .vsix file from repo 
2. Open cursor or vscode and cd into the location where the .vsix is 
3. Run the following command based on the IDE you are using 
    - cursor --install-extension gab-syntax-highlighting-0.0.1.vsix 
    - code --install-extension gab-syntax-highlighting-0.0.1.vsix 
4. One it is installed press Ctrl+Shift+P and type Preferences: Color Theme 
5. Select GAB Theme from the list
